Keyflow

2021 - Arduino, Java

NEXT

Keyflow, created on a team of two including myself and Joey Barnes, uses the pitch of audio inputs to create a unique visual on the screen. Each note has a different color and different type of stroke. The direction of each output is random, using perlin noise. Every encounter will produce a different result.
